# Exercícios de Python - Curso Gustavo Guanabara

Repositório criado para praticar conceitos básicos e intermediários de **Python**, seguindo o curso do Gustavo Guanabara. Contém diversos exercícios que ajudam a consolidar lógica de programação e sintaxe da linguagem.

## Tecnologias
- Python 3.x

## Conteúdo
- Variáveis e tipos de dados
- Estruturas condicionais (`if`, `else`, `elif`)
- Loops (`for`, `while`)
- Funções
- Listas e dicionários
- Exercícios comentados e soluções práticas

## Objetivo
Praticar **lógica de programação** e **fundamentos de Python** através de exercícios guiados pelo curso. Ideal para fixar aprendizado e criar uma base sólida para desenvolvimento em Python.

> Este repositório é voltado para **estudo e prática**, registrando a evolução do aprendizado.
